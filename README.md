# Тестовое задание TrueConf

## Задача

Написать программу, которая подсчитывает число TCP-пакетов в заданном .pcap-файле.

Условия:
1. Программа должна представлять собой утилиту командной строки и принимать путь к .pcap-файлу в качестве аргумента.
2. Программа должна быть написана на C или C++.
3. Возвращаемое программой значение должно быть равно количеству TCP-пакетов во входном файле.
4. Для парсинга .pcap программа должна использовать libpcap.
5. Сборка программы должна осуществляться под Linux с помощью утилиты make. 
6. Результат выполнения задания нужно прислать в виде ссылки на git-репозиторий на любом хостинге.
7. Для тестирования работоспособности программы рекомендуется записать дамп сетевого трафика с помощью Wireshark.

Бонус:
Добавить опции командной строки для фильтрации TCP по исходному(source) порту и порту назначения(destination). Выводить количество пакетов, попавших под условие.

## Сборка и тестирование

Установка библиотеки libpcap:
```
$ sudo apt install libpcap-dev
```

Сборка программы tcpcount:
```
$ make
```

Запуск тестов:
```
$ make test
```
